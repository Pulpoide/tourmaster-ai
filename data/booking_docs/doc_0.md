# Base de Datos de Bares y Clubes de Música en Córdoba, Rosario y Buenos Aires

## Introducción

Este documento técnico tiene como objetivo proporcionar un recurso útil para artistas, promotores y aficionados a la música que busquen espacios en Córdoba, Rosario y Buenos Aires para actuar o asistir a eventos en vivo. Esta primera parte incluye una lista de bares y clubes destacados en cada una de estas ciudades, con detalles sobre sus géneros musicales, capacidad y datos de contacto.

## Estructura de la Base de Datos

La base de datos está organizada en secciones por ciudad, y dentro de cada sección, los locales están clasificados por su nombre. Para cada local, se incluyen los siguientes campos:

- **Nombre**
- **Género Musical (Rock, Jazz, Indie, DJ)**
- **Capacidad**
- **Contacto (Teléfono, Email, Redes Sociales)**
