# Base de Datos de Bares y Clubes de Música en Córdoba, Rosario y Buenos Aires

## Introducción

La música en vivo es un pilar fundamental de la cultura argentina y las ciudades de Córdoba, Rosario y Buenos Aires ofrecen una variada gama de espacios donde se puede disfrutar de distintos géneros musicales. Este documento técnico tiene como objetivo proporcionar una base de datos con información relevante sobre bares y clubes musicales en estas tres ciudades. 

### Estructura de la Base de Datos

Cada entrada de la base de datos incluirá los siguientes campos:
- Nombre del local
- Género musical
- Capacidad
- Información de contacto
